基于jQuery的省市区三级联动插件，数据来源于国家统计局行政规划代码数据，真实有效。

作者：弦云孤赫(David Yang) Yang

调用省市区三级联动插件：

首先js文件夹里需包含json文件 — jquery-citys.json，
再引入插件js文件 — jquery-citys.js

注意:由于本页面包含的AJAX方法涉及到 跨域 问题，在本地测试时，须通过服务器环境运行，类似这样：
http://127.0.0.1:8888/index.html

默认参数初始化设置

$('#demo1').citys();

通过地区编码初始化设置

$('#demo2').citys({code: '510116'});

通过地区名称初始化设置，并且下拉框值为地区名称

$('#demo3').citys({type: 'name',province: '四川省',city: '成都市',area: '武侯区'});

常用参数调用示例


  Html:
  <div class="citys" id="citys">
    <select class="city-select" name="province"></select>
    <select class="city-select" name="city"></select>
    <select class="city-select" name="area"></select>
  </div>

  Javascript:
  $('#citys').citys(options);

options参数

参数 默认值 说明
url "js/jquery-citys.json" 省市区json数据地址，不可更改
selProvince "province" 省份、直辖市列表框的name
selCity "city" 城市列表框的name
selArea "area" 区、县列表框的name
type "code" 下拉框值的类型,code行政区划代码,name地名
code 0 地区编码
province 无 省份(省级),可以为地区编码或者名称
city 无 城市(地级),可以为地区编码或者名称
area 无 地区(县区级),可以为地区编码或者名称
